# API Test framework AXIOS + CUCUMBER

Tools :

[Cucumber](https://cucumber.io/) 

[Axios](https://github.com/axios/axios)

For assertions read on :
[Jest Matchers](https://jestjs.io/docs/using-matchers)

### Pre-requisites
Node v12+

### Install dependencies
```
npm install
```

## Run tests 

```
npm test
```

## Get report

```
npm run report
```
